# PomoTimer

A customizable Pomodoro timer built with React, TypeScript, and Tailwind
CSS.

## Features

-   Work, short break, and long break modes
-   Custom timer durations and long-break intervals
-   Auto-start options for breaks and Pomodoro sessions
-   Settings saved with `localStorage`
-   Responsive and accessible UI

## Tech Stack

React · TypeScript · Tailwind CSS · Vite

## Live Demo

Deployed on Vercel --- https://pomodoro-timer-ks.vercel.app/

## Run Locally

``` bash
git clone https://github.com/KunalSambyal/Pomodoro-Timer.git
cd Pomodoro-Timer
npm install
npm run dev
```
